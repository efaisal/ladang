ladang - High Level API¶

This module provides the a high level, more Pythonic interface to inotify as compared to the low level _ladang module, a low level C extension.

class ladang.Ladang(flags=0)¶

A wrapper class to _ladang to provide a more Pythonic interface to inotify.

close()¶

Close the control file descriptor of Ladang object.

get_event()¶

Fetch events on watched file or directory..

Each event will be represented as Python dictionary, with the following keys:

Key Meaning
wd Watched file of directory
mask Mask of events
cookie Unique cookie associating related events
name Optional null-terminated name

Returns a tuple of events.

get_fd()¶

Get the inotify event queue file descriptor.

Returns a file descriptor associated with inotify event queue.

unwatch(pathname)¶

Unregister watched file or directory.

watch(pathname[, mask=ladang.IN_ALL_EVENTS])¶

Register a file or directory to watch on event(s) defined by mask. (see Inotify event mask constants)

Inotify event mask constants¶

ladang.IN_ACCESS¶

File was accessed.

ladang.IN_MODIFY¶

File was modified.

ladang.IN_ATTRIB¶

Metadata changed.

ladang.IN_CLOSE_WRITE¶

Writtable file was closed.

ladang.IN_CLOSE_NOWRITE¶

Unwrittable file closed.

ladang.IN_CLOSE¶

Equivalent to ladang.IN_CLOSE_WRITE | ladang.IN_CLOSE_NOWRITE

ladang.IN_OPEN¶

File was opened.

ladang.IN_MOVED_FROM¶

File was moved from X.

ladang.IN_MOVED_TO¶

File was moved to Y.

ladang.IN_MOVE¶

Equivalent to ladang.IN_MOVED_FROM | ladang.IN_MOVED_TO

ladang.IN_CREATE¶

Subfile was created.

ladang.IN_DELETE¶

Subfile was deleted.

ladang.IN_DELETE_SELF¶

Self was deleted.

ladang.IN_MOVE_SELF¶

Self was moved.

ladang.IN_ONLYDIR¶

Only watch the path if it is a directory.

ladang.IN_DONT_FOLLOW¶

Do not follow a sym link.

ladang.IN_EXCL_UNLINK¶

Exclude events on unlinked objects.

ladang.IN_MASK_ADD¶

Add to the mask of an already existing watch.

ladang.IN_ISDIR¶

Event occurred against dir.

ladang.IN_ONESHOT¶

Only send event once.

ladang.IN_ALL_EVENTS¶

All events which can be waited on.

Equivalent to:

ladang.IN_ACCESS | ladang.IN_MODIFY | ladang.IN_ATTRIB | ladang.IN_CLOSE_WRITE | ladang.IN_CLOSE_NOWRITE | ladang.IN_OPEN | ladang.IN_MOVED_FROM | ladang.IN_MOVED_TO | ladang.IN_CREATE | ladang.IN_DELETE | ladang.IN_DELETE_SELF | ladang.IN_MOVE_SELF
